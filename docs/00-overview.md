# Lab Overview

This repository documents a Proxmox VE homelab running on a Minisforum UM890.

## Hardware
- Minisforum UM890

## Hypervisor
- Proxmox VE 9.1.4

## Host
- Hostname: pve-lab

## VM
- Name: openg2p
- OS: Ubuntu Server 22.04

## Goal
Provide an isolated, reproducible lab environment with controlled internet access
for OpenG2P services.
